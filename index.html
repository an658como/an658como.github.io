Minimal Social Media Application

This application was developed using a powerful stack of modern technologies including Python, FastAPI, PostgreSQL, SQLAlchemy, Alembic, and Docker. It represents a simple but scalable social media platform with essential functionalities for user and content management.

The app provides the following key endpoints:

  • Create a user: Allows new users to register.
  • Login: Authenticates users and provides them with secure tokens.
  • Create a post: Enables logged-in users to create new posts.
  • Get all posts: Enables logged-in users to see all public posts.
  • Get a post by ID: Enables logged-in users to see a specific post.
  • Delete a post: Allows users to remove their own posts.
  • Vote on a post: Gives users the ability to upvote or downvote posts.

Check out the source code for this project on GitHub: Minimal Social Media App - GitHub
